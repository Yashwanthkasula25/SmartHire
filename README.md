🚀 SmartHire – AI-Based Hiring Platform
📌 Overview

SmartHire is an AI-powered web application designed to simplify and automate the hiring process.
It analyzes candidate details and predicts the probability of selection using machine learning algorithms.

The platform helps recruiters make faster and smarter hiring decisions.

🎯 Key Features
🔐 User Authentication (Login & Registration)
📩 Email OTP Verification System
🤖 AI-Based Candidate Selection Prediction
📊 Displays Candidate Selection Probability
👨‍💼 Separate User & Admin Functionalities
📝 Input Candidate Details (Skills, Experience, etc.)
🌐 Responsive Web Design
🔒 Secure Data Handling

🧠 How It Works
User registers with OTP verification
User logs into the system
Enters candidate details
ML model processes the input data
System predicts hiring probability
Result is displayed to the user

⚙️ Installation & Setup
1️⃣ Clone Repository

cd SmartHire
2️⃣ Create Virtual Environment
python -m venv venv
venv\Scripts\activate
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Run Application
python app.py
5️⃣ Open in Browser
http://127.0.0.1:5000/
